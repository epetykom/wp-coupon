English
WordPress plugin, which generate picture with individual number or code.
Русский
Плагин WordPress, позволяющий генерировать изображение с индивидуальным кодом.
English
Originally based on VoucherPress plugin by Chris Taylor, but i make this awesome plugin smaller and simpler. I have some issue with cyrillic text. I just make to gen little jpg pictures instead of pdf buggy mosnters.
Русский
Изначально основан на плагине VoucherPress от Chris Taylor, но я зделал этот прекрасный плагин меньше и проще. У меня были проблемы с кирилическим текстом. Я просто переделал генерацию на маленькие jpg картинки вместо глчюных pdf монстров.

Install

English
  1. Extract archive to the `/wp-content/plugins/` directory and rename new directory to `wp-coupon`.
  2. Activate the plugin through the 'Plugins' menu in WordPress.
  3. Tune up setting of plugin in admin dashboard.
Русский
  1. Распакуйте архив каталог `/wp-content/plugins/` и переименуйте новую папку в `wp-coupon`.
  2. Активируйте плагин через меню 'Плагины' в WordPress.
  3. Настройте плагин в панели администратора.
