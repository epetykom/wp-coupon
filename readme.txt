== Description ==

Wordpress plugin, which generate picture with individual number or code.

May be need for business. You may give site visitors feature to generate 
personal disount coupon.

Originally based on [VoucherPress](http://wordpress.org/extend/plugins/voucherpress/ "VoucherPress plugin")
by Chris Taylor, but i make this awesome plugin smaller and simpler. 
I have some issue with cyrillic text. I just make to gen little jpg 
pictures instead of pdf buggy mosnters.

== Installation ==

1. Extract `indiflyer.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Tune up setting of plugin in admin dashboard
